Welcome to your first MDX blog post! This example demonstrates how to structure your blog content using MDX with frontmatter metadata.

## What is MDX?

MDX is a format that lets you seamlessly write JSX in your Markdown documents. You can import components, such as interactive charts or alerts, and embed them within your content.

## Frontmatter Metadata

At the top of each MDX file, you'll find a YAML frontmatter block (between the `---` delimiters) that contains metadata about the post:

- **title**: The post title displayed in headings and SEO
- **excerpt**: A brief description for post cards and previews
- **date**: Publication date in YYYY-MM-DD format
- **readTime**: Estimated reading time
- **categories**: An array of predefined categories
- **featured**: Whether to highlight this post
- **description**: SEO meta description (optional)
- **keywords**: SEO keywords array (optional)

## Using React Components

One of the powerful features of MDX is the ability to use React components directly in your content:

```jsx
import { Button } from '@abbottland/fui-components';
<Button color="primary">Click me!</Button>;
```

## Code Blocks

You can include syntax-highlighted code blocks:

```typescript
interface BlogPost {
  title: string;
  excerpt: string;
  date: string;
  categories: BlogCategory[];
}
```

## Rendering Diagrams

You can render interactive React Flow diagrams in your blog posts. The `DiagramViewer` component is available globally in all MDX files. Just import your diagram JSON file and use it:

```jsx
import diagramData from './diagram.json';

<DiagramViewer data={diagramData} />;
```

The diagram is interactive - you can pan and zoom to explore the nodes and connections. Diagrams are created using the diagram-maker app and exported as JSON files.

## Next Steps

To create a new blog post:

1. Create a new folder in `src/content/blog/` with your post slug as the folder name
2. Add an `index.mdx` file with frontmatter and content
3. The post will automatically be available at `/blog/[slug]`

Happy blogging!
